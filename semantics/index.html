Details Element

The details tag specifies additional details that the user can open and close on demand.

Use of Semantic Tags

These tags actually reinforce the meaning of the content displayed on the web page rather than merely define the appearance.

Section Element

Section element defines a section in the document. A section is a thematic grouping of content, typicaly with a heading. A section can be used for

  • Chapters
  • Introduction
  • News Items
  • Contact Information

Article Element

The article element specifies independent, self-contained content. An article should make sense on its own, and it should be possible to distribute it independently from the rest of the web site.

Article element can be use for

  • Forum Posts
  • Blog posts
  • User Comments
  • Product cards

Header-Footer Elements

Header: The header element represents a container for introductory content and a set of navigation links
Footer: The footer element represents footer of the document

Mark Element

This tag is used to define the text that has to be marked or highlighted

Aside Element

The aside tag defines some content aside from the content it is placed in. The aside content should be indirectly related to the surrounding content.

This content is often placed as a sidebar in a document
